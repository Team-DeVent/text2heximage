# TextToHexImage

텍스트를 16진수로 변환하여 hex color 이미지로 표현하는 파이썬 프로젝트 입니다. 또한 생성된 이미지를 텍스트로 변환할 수 있습니다.  

## Installation

```
pip3 install Text2HexImage
```

## Usage

```python
from text2heximage import core

control = core.T2i()
print(control.generatePrimaryId())
```
